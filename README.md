# Best Virtual - Expo

Projeto Expo (React Native) gerado a partir do seu componente React.

## Como usar

1. Coloque sua chave da Groq em `app.json` -> `expo.extra.GROQ_API_KEY` ou use variáveis de ambiente.
2. Instale dependências:

```bash
npm install
npx expo start
```

3. Abra o Expo Go no seu celular e escaneie o QR code.

## Observações
- O endpoint de histórico `API_BASE_URL` está definido em `App.tsx`. Ajuste para o seu backend.
